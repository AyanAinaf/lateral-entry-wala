# Lateral Entry Wala

A smart VTU exam preparation platform designed to help students prepare efficiently through previous year question paper analysis, important question extraction, formula revision sheets, and syllabus-based search tools.

## Features

### Previous Year Paper Analysis

* Year-wise VTU question paper collection
* Structured module-wise question extraction
* Question frequency analysis
* Important question identification

### Module Search Engine

* Search questions by topic, formula, keyword, or concept
* Filter questions by syllabus module
* Quick access to relevant exam questions

### Practice Test Builder

* Create custom practice sets from previous year questions
* Focus on high-probability exam topics
* Structured revision workflow

### Formula Reference Desk

* Subject-wise formula sheets
* Quick revision cards
* Exam-oriented equation reference

### Supported Subjects

* Electromagnetics (BEC401)
* Principles of Communication (BEC402)
* Control Systems (BEC403)
* Microcontrollers (BEC405A)
* Biology for Engineers (BBOC407)

## Technology Stack

* React
* TypeScript
* Vite
* CSS3
* Vercel

## Project Goal

The objective of Lateral Entry Wala is to provide VTU students with a centralized academic preparation platform that helps them:

* Analyze previous year examination patterns
* Identify frequently asked questions
* Revise important formulas quickly
* Prepare efficiently for semester examinations
